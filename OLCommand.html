An unordered list (ul) is used to create a list of items in no particular order i.e. the order of items is not relevant. By default, the items in this list will be marked with bullets. Whereas, an ordered list (ol) is used to create a list of items in a specific order.


The (a) HTML element (or anchor element), with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address. Content within each (a) should indicate the link's destination.


HTML table tag is used to display data in tabular form (row * column). There can be many columns in a row. We can create a table to display data in tabular form, using (table) element, with the help of (tr) , (td), and (th) elements.


The HTML form tag is required when you want to collect information that visitors provide. For example, you may want to collect specific data from visitors, such as name, email address, and password. The HTML (form) tag is used to create a form.


The (select) element is used to create a drop-down list. The (select) element is most often used in a form, to collect user input.


The (textarea) tag defines a multi-line text input control. The (textarea) element is often used in a form, to collect user inputs like comments or reviews.
